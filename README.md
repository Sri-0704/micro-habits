# Micro-Habits Garden
A gamified habit-tracking ecosystem where habits become growing plants.

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
